Organizations
Teams + orgs, built in

Create an organization, invite your team, and share a unified workspace — memory, workflows, and billing all in one account.

🏗️ Create an Organization

One click to create an org. Assign a plan, set the billing owner, and start inviting members.

📨 Invite by Email

Invite teammates via email. They get a secure token link — no password required to accept.

🔑 Role-Based Access

Owner, admin, and member roles. Admins manage billing and membership; members access shared workspace.

🧠 Shared Memory

Org-scoped persistent memory in Copilot. Every team member sees the same context and history.

📊 Usage Dashboard

See message counts, workflow runs, and API calls per member. Set limits before bills surprise you.

💳 Unified Billing

One Stripe subscription covers the whole org. Upgrade, downgrade, or cancel from the portal in seconds.

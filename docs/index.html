Quickstart

Up in 60 seconds

Clone, install deps, start the backend. The WebUI opens at http://127.0.0.1:8001.

# 1. Clone git clone https://github.com/andrewembry312-hub/elabs-platform.git cd elabs-platform # 2. Python env python -m venv .venv # Windows: .venv\Scripts\Activate.ps1 # Linux: source .venv/bin/activate pip install -r backend/requirements.txt # 3. Configure cp .env.example .env # Edit .env — set ELABS_JWT_SECRET # 4. Start .\scripts\start.ps1 # Windows ./scripts/start.sh # Linux / macOS

To expose www.elabs.com and gateway.elabs.com with auto-TLS, run Caddy.

# Install Caddy (Windows) # https://caddyserver.com/docs/install # Start (Linux/macOS — requires DNS pointed first) caddy run --config caddy/Caddyfile # Local dev (no DNS, no TLS, any OS) caddy run --config caddy/Caddyfile.local # Then add to /etc/hosts (or Windows hosts): 127.0.0.1 www.elabs.com 127.0.0.1 gateway.elabs.com # Verify curl http://www.elabs.com:8888/

⚡ THE MACHINE — Help

THE MACHINE is a multi-agent workflow engine. It decomposes complex prompts into subtasks, assigns each to a specialist agent, and executes them in sequence with context passing.

How It Works

1. Prompt — You write a complex request in the chat or directly in the Core Prompt editor.

2. Decomposition — The CEO analyzes your prompt and breaks it into ordered subtasks.

3. Assignment — Each subtask is assigned to the optimal specialist agent (muscle).

4. Execution — Click ▶ RUN ALL to execute sequentially, or ⏭ STEP to run one node at a time.

5. Results — Each agent's output feeds into the next. Files appear in the Workspace panel.

Tips

• Be specific in your prompt — the more detail, the better the decomposition.

• Each agent gets context from previous completed nodes — they build on each other.

• Use ↻ RESET to refine your prompt without losing the project.

• Files created during execution appear in the Workspace panel and persist on disk.
